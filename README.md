# PersonalHomeService

Spring Boot 个人服务项目。

## 技术栈

- Java 23
- Spring Boot 3.2.3
- Maven

## 项目结构

```
PersonalHomeService/
├── pom.xml                 # Parent POM
└── app-starter/            # 启动模块
    ├── pom.xml
    └── src/main/java/      # 启动类
```

## 运行

```bash
mvn spring-boot:run -pl app-starter
```

## 开发

本项目使用 [OpenSpec](https://github.com/charleswang1999/openspec) 进行变更管理。
